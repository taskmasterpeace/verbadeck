### **For Pexels Application:**

```
VerbaDeck is an AI-powered voice-controlled presentation platform that helps
users create and deliver professional presentations hands-free.

How we'll integrate Pexels photos:

1. AI CONTENT ANALYSIS: When users create presentation slides, our AI analyzes
   the slide content (e.g., "Team Culture", "Revenue Growth", "Product Launch")
   and generates smart search queries.

2. IMAGE RECOMMENDATIONS: We'll use the Pexels API to search for 4 curated
   photo options per slide, displayed in a grid with photographer attribution.

3. ONE-CLICK INSERTION: Users can click any recommended image to instantly
   insert it into their presentation slide.

4. PRESENTATION DISPLAY: Images are displayed alongside slide text in our
   dual-monitor presenter mode (presenter view + audience view).

Use Case: Educational and business presentations, conference talks, sales
pitches, training materials.

Attribution: Photographer credits are displayed in our image selection dialog
and preserved in the presentation metadata.

Estimated usage: 50-200 API requests per day across our user base.
```

### **For Unsplash Application:**

```
VerbaDeck is an AI-powered presentation platform that enables voice-controlled
slide advancement for hands-free presenting.

How we'll integrate Unsplash photos:

INTEGRATION POINTS:
• In our slide editor, users can click "AI Image Recommendations"
• Our AI analyzes slide content and generates contextual search queries
• We use Unsplash API to fetch 4-6 curated photo options
• Users select images via visual gallery with quality indicators
• Selected images are inserted into presentation slides

WHERE PHOTOS APPEAR:
• Slide editor (preview and selection interface)
• Presenter view (speaker's display with notes and controls)
• Audience view (clean full-screen display for projectors/second monitors)
• Exported presentations (saved as part of .verbadeck files)

ATTRIBUTION:
• Photographer name and profile link shown in selection dialog
• Download tracking via Unsplash API (as per guidelines)
• Metadata preserved in presentation files

TARGET USERS: Educators, business professionals, conference speakers, sales teams

ESTIMATED VOLUME: 100-500 requests per day

APPLICATION TYPE: Open-source educational tool for improving presentation
quality and accessibility.
```
