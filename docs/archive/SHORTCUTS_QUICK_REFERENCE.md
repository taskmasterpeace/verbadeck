# VerbaDeck Keyboard Shortcuts - Quick Reference

## Help
| Shortcut | Action |
|----------|--------|
| **Ctrl+/** | Show keyboard shortcuts help |

## File
| Shortcut | Action |
|----------|--------|
| **Ctrl+S** | Save presentation |
| **Ctrl+O** | Open presentation |
| **Ctrl+N** | New presentation |
| **Ctrl+L** | Open library |

## Navigation
| Shortcut | Action |
|----------|--------|
| **Ctrl+H** | Go to home/create |
| **Ctrl+E** | Focus editor |
| **Ctrl+P** | Start presenter mode |
| **Ctrl+K** | Open Know It All |

## Editor
| Shortcut | Action |
|----------|--------|
| **Ctrl+T** | Test triggers |
| **Ctrl+Q** | Knowledge base |

## Voice
| Shortcut | Action |
|----------|--------|
| **Ctrl+Space** | Toggle voice control |
| **Ctrl+M** | Toggle Q&A mode |

---

💡 **Tip**: Press **Ctrl+/** anytime to see the full interactive shortcuts help.

📝 **Note**: On Mac, use **⌘ Cmd** instead of **Ctrl**.
